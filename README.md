桌面时钟 (HorizontalClock)

一个功能丰富的Android桌面时钟应用，完美支持从Android 4.1到Android 16的所有版本，同时兼容TV端和触屏设备。

📱 项目特色

🌟 核心功能

· 精美的时间显示 - 大字体时间、日期、农历、节日信息
· 实时天气 - 自动获取当前位置天气信息
· 智能壁纸 - 自动更换精美风景壁纸
· 音乐氛围灯 - 检测音乐播放时显示彩色氛围灯效果
· 多设备支持 - 同时兼容手机、平板、TV电视端

🎨 个性化定制

· 颜色主题 - 9种文字颜色、7种背景颜色随意切换
· 字体样式 - 4种不同字体风格可选
· 手势操作 - 单击换色、双击换字体、滑动设置位置
· TV遥控器支持 - 完整的方向键和OK键操作

📺 TV端特别优化

· 焦点导航 - 完善的焦点指示器和导航逻辑
· 遥控器操作 - 上下键换颜色、左右键换字体、OK键操作
· 横屏适配 - 完美的TV横屏显示体验
· 双击返回退出 - 防止误操作

🛠 技术特点

兼容性保障

· 最低支持Android 4.1 (API 16)
· 目标版本Android 16 (API 36)
· 使用Support Library确保向后兼容
· TV Leanback支持

性能优化

· 内存高效管理 - 智能壁纸缓存和回收
· 网络优化 - 移动数据使用提醒和流量控制
· 电池友好 - 合理的更新频率和资源释放

📋 功能详情

时间信息显示

· 实时时钟（时:分:秒）
· 完整日期（年月日+星期）
· 农历日期和传统节日
· 节气提醒

天气服务

· 自动定位（省市区三级选择）
· 温度、天气状况显示
· 天气图标可视化
· 多数据源备份确保稳定性

壁纸系统

· 自动从网络获取精美壁纸
· 本地缓存10张壁纸循环使用
· 每5分钟自动更换
· 支持手动立即更换

音乐氛围灯

· 实时检测音乐播放状态
· 左右两侧彩色渐变灯光
· 呼吸灯效果，随音乐节奏变化
· 40+种颜色自动轮换

电池信息

· 实时电池电量显示
· 充电状态指示
· 自适应颜色匹配

🔧 安装使用

系统要求

· Android 4.1 (API 16) 或更高版本
· 横屏显示设备效果最佳
· 网络连接（用于天气和壁纸）

首次启动流程

1. 显示操作说明
2. 关于开发者信息
3. 位置设置（用于天气服务）
4. 进入主时钟界面
